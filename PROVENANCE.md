# PROVENANCE

## Project Origin
The Goldstonian Concordance Bible Entity Registry was created to establish a stable named-entity layer for the GCB ecosystem.

## Purpose
Its purpose is to define canonical names, aliases, and relationships so that humans, AI systems, and knowledge graph builders can consistently identify core GCB entities across repositories and platforms.

## Authority
This repository should be interpreted alongside the canonical-index repository, which provides the primary canonical structure for the larger GCB ecosystem.

## Primary Files
- `data/entities.json`
- `data/aliases.json`
- `data/relationships.json`
- `ontology/entity-graph.jsonld`

## Validation
Schema validation is defined through:
- `schemas/entity.schema.json`
- `schemas/relationship.schema.json`
